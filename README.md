# Getting Started with Calendar

實作 task1
1. 第一次點擊設定起始日
2. 下次點擊日期與當前選擇選項相同或晚於當前選項將其設置為結束日。
3. 下次點擊日期早於當前選項將重置起始日。(文件沒有說明是否要重置日期, 最後都會被選擇)

### `npm i`
安裝所需套件

### `npm run dev`
啟動開發環境, [http://127.0.0.1:3000](http://127.0.0.1:3000)

### `npm test`
執行 jest 測試,

### `npm run build`
建置正式環境的打包程式碼

### `npm run prod`
起動正式環境, [http://127.0.0.1](http://127.0.0.1)
